# 🏡 农庄卡牌：田园物语
**Farm Cards: Pastoral Tale** | 克苏鲁风格肉鸽卡牌种田游戏

![游戏截图](docs/screenshot.png)

---

## 🎮 游戏特色
- 🎴 **120张全克苏鲁风格立绘卡牌**：作物、动物、工具、建筑、事件五大类，全部由AI生成专属腐化立绘
- 🎲 **肉鸽随机玩法**：随机牌库、随机事件、随机商店，每局体验都不同
- ⚡ **组合技系统**：特定卡牌搭配触发强力组合效果，策略性拉满
- 🏆 **成就系统**：50+成就等你解锁，包含普通/稀有/史诗/传说/隐藏成就
- 🌤️ **随机事件系统**：天降大雨、小偷来袭、女神祝福等随机事件改变战局
- 📈 **成长系统**：卡牌升级、难度选择、遗物收集，多维度成长路径

## 🎯 核心玩法
1. **抽卡**：每回合抽取手牌，包含作物、动物、工具等不同类型卡牌
2. **打牌**：消耗能量拖放卡牌到游戏区，获得对应资源和效果
3. **经营**：种植作物、养殖动物、建造建筑，积累资源升级卡牌
4. **闯关**：完成每日金币目标，推进游戏阶段，解锁更多卡牌和内容
5. **组合**：搭配特定卡牌触发组合技，获得额外增益效果

## 🛠️ 技术栈
- 前端：原生JavaScript + Canvas 2D渲染，无第三方框架，轻量高效
- 立绘：字节跳动豆包Seedream API生成全120张克苏鲁风格卡牌立绘
- 部署：GitHub Pages 一键部署，在线即玩
- 架构：ECS（实体组件系统）架构设计，扩展性强
